# build and submit this project

**Todo List Application (Terminal)**
Build a todo list application using arrays and objects that runs in Node.js:
- Store todos as an array of objects
- Add, remove, and list todos
- Mark todos as complete
- Save todos to a JSON file (optional)
- Use readline for user interaction
- Run with `node todo.js`

**Requirements**
- Todos stored as an array of objects (e.g., `{ id, text, done }`). Use only concepts from Modules 1â€“4 (variables, control flow, functions, arrays, objects, readline).
- Support at least: add todo, list todos, mark complete/incomplete, remove todo. Use readline (or similar) for menu/input.
- Program runs with `node todo.js`. Optional: persist to a JSON file using `fs` (only if taught) or skip for full credit.

**Submission**
Submit your `todo.js` (and optional data file). The app should run without errors.
